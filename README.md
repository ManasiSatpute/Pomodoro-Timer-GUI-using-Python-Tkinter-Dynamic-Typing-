A simple and user-friendly Pomodoro Timer desktop application built using Python and Tkinter.
This app helps improve focus and productivity by following the Pomodoro Technique, which alternates focused work sessions with short and long breaks.

The Pomodoro Technique is a time management method that encourages working in focused intervals followed by breaks.
This application provides a clean graphical interface that automates these sessions, making it easier to stay productive without manually tracking time.

Features:
1. Work, Short Break, and Long Break sessions
2.  Real-time countdown timer
3. Automatic session switching
4. Simple and intuitive Tkinter GUI
5. Lightweight and easy to run
